# Проект: Wine-shop

# О проекте: 
Адаптивная главная страница винного магазина.

# В проекте использованы:
* HTML разметка;
* БЭМ структура;
* CSS свойства;
* Flexbox;
* Grid;
* Резиновая верстка;
* Адаптивная верстка.
